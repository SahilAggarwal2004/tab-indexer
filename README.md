# Tab Indexer: Redefine How You Tab Through the Web

Make tab navigation on any website smooth and customizable with **Tab Indexer**. This extension lets you control the tab order of elements, so you can move through webpages exactly the way you want.

### Features

- **Custom Tab Order:** Define the order in which elements are focused when you press `Tab`.
- **Element Selection Mode:** Click on elements to include them in your custom tab order.
- **Automatic Updates for SPAs:** Works seamlessly with single-page applications.
- **Visual Feedback:** Highlights elements while selecting them.
- **Save Configurations:** Keep your tab order settings for different pages.

### Keyboard Shortcuts

Quickly manage Tab Indexer using these shortcuts:

- Press **Ctrl + Shift + A** to enable/force apply config
- Press **Ctrl + Shift + D** to disable extension
- Press **Escape** to exit element selection mode
